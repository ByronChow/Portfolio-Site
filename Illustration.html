THE CYREN PROJECT
The Cyren Project is a visual amalgamation of the research done during my Proxy thesis project. Using illustrations as a foundation, each element is treated as a collage rather than a finalized piece. Thinking about the project within the context of modern branding practices, the end goal of this project is to create a visual language based around intricate illustrative renderings.
